# Green Timer

A Trackmania-inspired timer mod for Beton Brutal that displays a green timer indicator showing your current run performance compared to your personal best.

## Description

Green Timer is a performance tracking mod for Beton Brutal that provides real-time feedback during your runs. 

This mod seamlessly integrates into the game's UI and provides customizable display options through the BBModMenu settings interface, allowing you to adjust the timer's appearance to match your preferences.

## Technologies

- **Unity UI Toolkit** - Modern UI framework for in-game display
- **MelonLoader** - Mod loader framework for Unity games
- **BBModMenu** - Mod settings management library
- **C#** - Core programming language

## Installation

### Prerequisites

- Beton Brutal game installed
- [MelonLoader](https://github.com/LavaGang/MelonLoader) installed and configured
- [BBModMenu](https://github.com/MiaouZart/BBModMenu/releases) installed (required dependency)

### Steps

1. Download the latest release from the [releases page](https://github.com/MiaouZart/BBModGreenTimer/releases)
2. Extract the DLL file to your `BETON BRUTAL\Mods` folder
3. Ensure BBModMenu is installed in the same folder
4. Launch the game - Green Timer will automatically initialize
5. Configure your settings through the Mod Menu in the game's settings
